# Guía A1: Describir Personas en Español

Esta es una guía interactiva diseñada para el canal **Español Para Todos**. Está optimizada para dispositivos móviles y enfocada en el nivel A1 del MCER.

## Contenido
- 13 fichas pedagógicas sobre descripción física, gramática y vocabulario.
- Diseño interactivo con Glassmorphism.
- Estructura mobile-first.

## Créditos
- Canal: [Español Para Todos](https://www.youtube.com/@Espanolparatodos-channel)
- Nivel: A1
